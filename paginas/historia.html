  1. Antecedentes

    2. El precursor de Flash es un software de dibujo vectorial publicado por FutureWave Software, que se presentó con dos nombres:

    • SmartSketch
      • Versión para el SO PenPoint
      • Versión para los SO Windows y MacOS
    • FurureSplash
  2. Macromedia

  3. En noviembre de 1996 FutureSplash es adquirida por Macromedia, que presenta Flash 1.0. Esta herramienta está compuesta por:

    Macromedia
    Un editor de gráficos, que permite realizar animaciones con ellos
    Macromedia Flash Player
    Un visualizador para dichos gráficos

    Sus versiones más destacadas son:

    • Flash 5.0. Presenta la primera versión del lenguajeActionScript
        • Flash MX 2004. Presenta ActionScript 2.0, que incluye:
          • Programación orientada a objetos
          • Componentes para una UI mejorada
          • Toda una serie de caractarísticas avanzadas
        • Flash 8. Última versión de Macromedia
  4. Adobe

  5. Adquiere Macromedia en diciembre de 2005, quedándose con toda su línea de herramientas gráficas, que incluye, además de Flash:

    • Dreamweaver
        • Director
            • Fireworks
                • Authorware

                  Las versiones más destacadas de Flash en esta nueva andadura son:

                  1. Adobe Flash CS3 Professional, que presentaActionScript 3.0
                  • Adobe Flash CS4, que incorpora:
                    • Mejoras en el editor de Flash como:
                      • Cinematica inversa
                      • Animacion 3d básica
                      • Animacion basada en objetos
                    • Motor 3D, que permite transformaciones tridimensionales básicas
                  • Adobe Anime, que realiza la transición hacia la generación de contenido HTML5
